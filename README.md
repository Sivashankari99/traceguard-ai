# TraceGuard AI

*Intelligent Engineering Change Impact & Compliance Assistant*

TraceGuard AI is a RAG-powered AI assistant designed to explore how Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) can support engineering change impact analysis and configuration management compliance.

The system analyzes proposed engineering changes against a synthetic automotive engineering knowledge base containing requirements, configuration items, historical change requests, and baseline information. It retrieves potentially relevant engineering artifacts and uses an LLM to generate an explainable impact assessment for human review.

## 🚀 Project Goals

- Identify potentially impacted engineering requirements from a proposed change request.
- Retrieve related configuration items and historical change requests.
- Identify potential impacts on active baselines.
- Support configuration management compliance checks.
- Generate structured and explainable AI-assisted impact assessment reports.

## 📊 Dataset

This project uses entirely synthetic automotive engineering data created specifically for educational and portfolio purposes.

No proprietary, confidential, employer-specific, or real-world organizational data is used in this project.

## ⚠️ Disclaimer

TraceGuard AI is an educational and portfolio project. AI-generated impact assessments are intended to assist human analysis and should not be considered authoritative engineering or compliance decisions.
